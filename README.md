Escape route is a 3D game where you control a helicoptter, avoid obstacles and collect coins!

## Screenshot:
<img src="Assets/escape route GIF.gif" width ="250"/>

## Game logic:
Fly the hellicopter using arrow keys and avoid bolders, beat your best score and last longer!

## Tools used:
* Visual studio (IDE)
* C# (logic)
* Unity (game engine)
* Paint.Net (Image editting)
